# 📚 Study Verse

Study Verse is a comprehensive self-study educational platform for students from Class 1 to Class 12.

## Features
- 🎓 Student Registration & Login
- 📖 Old NCERT and New NCERT Support
- 🎥 Video Lectures
- 📝 Revision Notes
- ✍️ Practice Tests & Quizzes
- 🎮 Educational Learning Games
- 📊 Progress Tracking
- 🔔 Owner Announcements & Updates
- 📚 Multiple Subjects Across All Classes
- 📱 Mobile-Friendly Design

## Goal
To provide a complete learning ecosystem where students can learn, practice, revise, and improve their academic performance through interactive and engaging educational content.

## Platform Name
**Study Verse**
